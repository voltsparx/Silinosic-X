# Code of Conduct

## 1. Purpose

Silinosic-X is maintained as a proprietary security project. Participation in this
repository is expected to stay respectful, lawful, and aligned with the
project's stated license and authorized-use boundaries.

## 2. Expected behavior

Examples of behavior that contribute to a healthy project space:

- Being respectful in language, tone, and technical disagreement.
- Giving constructive, evidence-based feedback.
- Accepting accountability and correcting mistakes.
- Focusing on code, documentation, usage, and security concerns rather than
  personal attacks.
- Respecting legal, ethical, and consent-based boundaries for OSINT work.

Examples of unacceptable behavior:

- Harassment, intimidation, threats, discriminatory language, or abuse.
- Personal attacks, insults, trolling, or sustained disruption.
- Publishing private information without explicit permission.
- Encouraging illegal activity, unauthorized targeting, or abusive surveillance.
- Any behavior that conflicts with the repository license, stated usage limits,
  or applicable law.

## 3. Scope

This Code of Conduct applies to:

- repository discussions, issues, pull requests, and reviews
- project chat channels or community spaces linked to this repository
- public interactions where an individual is representing this project

## 4. Enforcement

Project maintainers are responsible for clarifying and enforcing this policy.

Maintainers may take any appropriate action in response to violations,
including:

- warning the contributor
- removing comments, issues, or pull requests
- temporarily or permanently restricting participation

Because Silinosic-X is proprietary software, maintainers also reserve the right to
deny participation or contributions that conflict with the project's license,
security posture, or authorized-use requirements.

## 5. Reporting

To report unacceptable behavior, contact:

- voltsparx@gmail.com

Please include relevant context, links, and timestamps where possible.

Reports will be reviewed in good faith and handled as promptly and
confidentially as practical.

## 6. Relationship to the license

This Code of Conduct does not replace the project license. The license controls
how Silinosic-X may be used, copied, or redistributed. This policy governs how
people are expected to behave while participating in the project and its public
spaces.
