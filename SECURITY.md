# Security Policy

## 1. Supported Versions

The latest stable version of Silinosic-X receives security updates.  
Older versions may not be actively maintained.

Please ensure you are using the most recent release before reporting issues.

---

## 2. Responsible Use

Silinosic-X is an intelligence and OSINT framework intended for:

- Personal research
- Educational purposes
- Authorized security assessments
- Defensive analysis

Users are responsible for ensuring they have proper authorization before scanning, analyzing, or interacting with any target.

Unauthorized use, abuse, or illegal activity is strictly discouraged and not supported by this project.

---

## 3. Reporting Vulnerabilities

If you discover a security vulnerability in Silinosic-X itself, please report it responsibly.

Contact:
- voltsparx@gmail.com

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Version number
- Relevant logs or screenshots

Do not publicly disclose vulnerabilities before allowing reasonable time for review and resolution.

---

## 4. Disclosure Policy

Upon receiving a report:

- The issue will be reviewed in good faith.
- Acknowledgment will be provided where possible.
- Fixes will be prioritized based on severity.
- Public disclosure (if appropriate) will occur after remediation.

---

## 5. Scope

This policy applies to vulnerabilities in the Silinosic-X framework codebase.

It does not apply to:
- Third-party tools
- External services
- Targets analyzed using Silinosic-X
